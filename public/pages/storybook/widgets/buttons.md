# Buttons

## Button types

```html
<nu-flex flow="row wrap" gap>
  <nu-btn>
    Base button
  </nu-btn>
  <nu-btn special>
    Special button
  </nu-btn>
  <nu-btn disabled>
    Disabled button
  </nu-btn>
  <nu-btn disabled special>
    Disabled special button
  </nu-btn>
  <nu-btn role="checkbox">
    Checkbox button
  </nu-btn>
</nu-flex>
```
