# Number inputs

Currency input:

```html
<preview/>
<nu-numinput
  prefix="$" value="12032" width="8" precision="2"
  label="Number input"></nu-numinput>
```

Percent input:

```html
<preview/>
<nu-numinput
  suffix="%" value="0" min="0" max="100" precision="0" width="8"
  label="Percents input"></nu-numinput>
```
